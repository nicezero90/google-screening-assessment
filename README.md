# Google Screening Assessment

This repository contains solutions for the Google Python & AI System Design take-home assessment.

## Project Structure

```
google-screening-assessment/
├── section-a-python/           # Python coding questions
├── section-b-ai-agent/         # AI Agent RAG system
│   ├── retrieval_agent/        # Handles data ingestion and insertion
│   ├── report_agent/           # Generates insights and reports
│   └── coordinator/            # MCP-style coordinator
├── data/                       # Sample data files
├── tests/                      # Unit tests
└── README.md                   # This file
```

## Sections

### Section A - Python Coding
- **Question 1**: List concatenation without '+' or 'extend()'
- **Question 2**: Base-N string to Base-10 number conversion

### Section B - AI Agent Coding (RAG)
- **Question 3**: Returns & Warranty Insights system with 2-Agent MCP-style architecture

## Setup Instructions

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Follow individual section instructions in their respective folders
